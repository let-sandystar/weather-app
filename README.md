# 🌦️ Weather app
A simple weather application originally built as a school assignment focused on API integrations. The initial version was created using HTML, CSS and TypeScript, with data fetched from external APIs to practice asynchronous requests, error handling, and basic UI updates.
I am now using this project as a learning playground for React, gradually refactoring the codebase to explore component-based architecture, state management, and more maintainable patterns. The goal is to transform a beginner-friendly assignment into a cleaner, more scalable application while documenting the learning process along the way.

## ✨ Features
- Search for cities and view current weather data
- API integration for weather and location data
- Responsive layout
- Ongoing refactor from vanilla TS to React

## 🛠️ Tech Stack
### Original version
- HTML
- SASS
- TypeScript
- Fetch-based API integration
### Current refactor
- React
- TypeScript
- Tailwind CSS
- Component extraction and UI/UX improvements

## 🏁 Project Goals
- Practice real-world API integration
- Learn and apply React fundamentals
- Improve structure, readability, and maintainability
- Explore UI/UX refinements and accessibility
- Document the evolution from a school assignment to a more polished project

## Setup
Install dependencies and run development server
```bash
npm install
```
